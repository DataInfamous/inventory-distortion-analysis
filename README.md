# inventory-distortion-analysis
Operations analytics project modeling inventory distortion caused by SSCC mislabeling in automated receiving Includes synthetic data set, python analysis, and Power BI dashboard quantifying correction cost and operational impact. 

## Project Overview
This project models inventory distortion caused by SSCC mislabeling during automated warehouse receiving.

## Dataset
The dataset contains synthetic shipment records including:
- shipment_id
- sku_id
- site_type
- error_type
- quantity discrepancies
- financial distortion

## Example Analysis
The dataset can be used to:
- quantify operational inventory distortion
- estimate financial loss caused by receiving errors
- build dashboards in Power BI
- perform analysis using Python and pandas
